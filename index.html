Help & FAQ

Quick answers to the questions we hear most often.

Last updated: 2026-07-02

Permissions

Why does Melo ask for Screen Recording?

Screen Recording is required to capture a region of your screen — both for screenshots with annotation and for OCR (reading text out of an image). macOS treats this as a sensitive capability, so you must approve Melo in System Settings → Privacy & Security → Screen Recording. The first time you trigger a capture, macOS shows its own permission sheet and Melo does not open System Settings until you tap the system sheet's “Open System Settings” button.

Why does Melo ask for Accessibility?

Only when you ask Melo to paste a clipboard item into another app. Apple requires this permission to be granted per-app in System Settings → Privacy & Security → Accessibility. Without it, Melo will still copy items to your clipboard and you can paste them manually with ⌘+V. See the Auto-Paste guide for setup steps.

Why does Melo ask for Microphone?

Only the Transcribe feature records audio from your microphone to convert it into text. If you don't use Transcribe, you don't need to grant Microphone. You can revoke it at any time in System Settings → Privacy & Security → Microphone and Melo will gracefully disable Transcribe.

App behaviour

Where is my clipboard history stored?

On your local Mac only, in a SQLite database under ~/Library/Application Support/Melo/. It is never uploaded to a server. You can clear it from Settings → Clipboard → Clear History.

Can I use Melo without an internet connection?

Yes. Every feature except StoreKit receipt verification works fully offline. The Storefront only needs connectivity when you start a subscription, restore purchases, or verify the trial window.

The first capture attempt didn't work — what now?

If you tapped “Allow” on Melo's in-app prompt but never tapped the system permission sheet's “Open System Settings” button, the system window closed without granting access. Trigger the capture again; this time Melo will take you straight to System Settings because it knows you've already seen the prompt once.

Subscription

How does the free trial work?

The first time you install Melo Suite from the App Store you get 14 days of full access. After the trial, single-feature subscriptions unlock individual tools (OCR, Clip, Capture, Color Picker) or the bundle subscription unlocks everything.

How do I restore my purchase on a new Mac?

Open Settings → Storefront → Restore Purchases and sign in with the Apple ID you used for the original purchase.
