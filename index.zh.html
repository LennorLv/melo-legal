权限

为什么 Melo 要请求屏幕录制？

屏幕录制是捕获你屏幕上某一块区域的必要权限——截图标注和 OCR（从图片里读文字）都需要。macOS 把这视为敏感能力，你必须在 系统设置 → 隐私与安全性 → 屏幕录制 里批准 Melo。第一次触发截图时，macOS 会自己弹出授权 sheet，Melo 在你点击系统的“打开系统设置”按钮之前不会替你跳设置。

为什么 Melo 要请求辅助功能？

只有在你让 Melo 把剪贴板内容粘贴到其他 app 时才需要。Apple 要求这个权限必须在 系统设置 → 隐私与安全性 → 辅助功能 里逐个 app 授权。没有授权时，Melo 仍然会把内容复制到剪贴板，你只需要手动按 ⌘+V 即可。具体开启步骤见 自动粘贴说明。

为什么 Melo 要请求麦克风？

只有“语音转文字”功能需要录制麦克风音频。如果你不用这个功能，可以不给麦克风权限，随时在 系统设置 → 隐私与安全性 → 麦克风 撤销，Melo 会自动关闭语音转文字。

app 行为

我的剪贴板历史存在哪里？

只存在你这台 Mac 本地，在 ~/Library/Application Support/Melo/ 下的一个 SQLite 数据库里，不会上传到任何服务器。你可以随时在 设置 → 剪贴板 → 清除历史 里清空。

断网时能用 Melo 吗？

可以。除 StoreKit 收据校验外，所有功能都完全离线可用。订阅商店仅在开通订阅、恢复购买或校验试用窗口时才需要联网。

第一次截图没成功，怎么办？

如果你在 Melo 的提示框点了“允许”，但没点 macOS 系统 sheet 上的“打开系统设置”按钮，系统 sheet 会关掉，但没有真正授权。再次触发截图，这次 Melo 已经记录“你看过一次系统 sheet”，会直接带你进系统设置。

订阅

免费试用怎么算？

从 App Store 首次安装 Melo Suite 起，你有 14 天完整试用。试用结束后，可以选择单功能订阅（OCR / Clip / Capture / Color Picker），也可以订阅 Melo Suite 全家桶。

新 Mac 上怎么恢复购买？

打开 设置 → 订阅商店 → 恢复购买，用当初购买时使用的 Apple ID 登录即可。
